🔐 CS50 Cybersecurity – Lecture 3: Securing Software

This repository contains my notes and practical understanding of Lecture 3: Securing Software from CS50’s Introduction to Cybersecurity.

---

📌 Topics Covered

- Phishing Attacks
- Code Injection (XSS)
- Reflected & Stored Attacks
- Character Escaping
- HTTP Security Headers
- SQL Injection
- Prepared Statements
- Command Injection
- Developer Tools Risks
- Server-Side Validation
- CSRF (Cross-Site Request Forgery)
- Arbitrary Code Execution (ACE)
- Open vs Closed Source Software
- App Stores & Package Managers
- Bug Bounty Programs
- Identifying Vulnerabilities (CVE, CVSS, EPSS, KEV)

---

🎯 Key Takeaways

- Never trust user input
- Always use server-side validation
- Use prepared statements to prevent SQL injection
- Escape special characters to prevent XSS
- Use security headers like CSP
- Implement CSRF tokens for protection
